Installation Instructions
Step 1

Copy and paste the following code anywhere between your <HEAD> and </HEAD>
tags in your page:

---
<script type="text/javascript" src="http://www.haloscan.com/load/cowtowncoder">
</script>
---

Step 2: Automated for Thingamablog Users

Copy and paste the following code into your template where the comments link should appear:

---
<a href="javascript:HaloScan('<$EntryID$>');" target="_self"><script
type="text/javascript">postCount('<$EntryID$>'); </script></a> | <a href="javascript:HaloScanTB('<$EntryID$>');" target="_self"><script type="text/javascript">postCountTB('<$EntryID$>');
</script></a>
---

Step 3

We would appreciate it if you could place a link to us somewhere on your page. The following code will result in a small button-link (just copy and paste the code anywhere in your page):


---
<a href="http://www.haloscan.com/"><img width="88" height="31" src="http://www.haloscan.com/halolink.gif" border="0"
alt="Weblog Commenting and Trackback by HaloScan.com" /></a>
---
